清华大学“未来动漫”兴趣团队是清华大学团委科创中心成立的动漫游戏研究创作团队。
团队旨在进行动漫游戏研究与创作，并培养动漫游戏专业创新人才。
团队成员由来自清华的15个不同院系的同学组成，从理工、人文社科、美术等不同领域而来，志在创作属于未来、属于自己的动漫游戏作品。
